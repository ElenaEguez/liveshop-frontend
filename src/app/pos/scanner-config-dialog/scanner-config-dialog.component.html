info_outline ¿Cómo usar un escáner físico?

Los escáneres USB y Bluetooth funcionan como teclados HID: envían el código como pulsaciones de teclas y presionan Enter automáticamente.

1
Conecta tu escáner USB al computador
O vincula el escáner Bluetooth con este dispositivo.
2
Haz clic en el campo de búsqueda del POS
El indicador verde confirma que está listo para escanear.
3
Apunta el escáner al código de barras del producto
El código se llenará automáticamente y buscará el producto.
4
Si hay 1 resultado exacto, se agrega solo al carrito
Si hay múltiples resultados, elige manualmente de la lista.
